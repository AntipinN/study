README

Программа простая как пробка.

Схема:
"Играть" <-> "Тип игры' <-> "Сложность" <-> "Игровое поле" -> "Окно результатов"
"Инструкция" <-> "Слайдер меню с текстом и дробями"
"Настройки" <-> "Выбор размера текста, звука, музыки, темы. Применение и По умолчанию" - фактически реализовано лишь изменение размеров текста, применение настроект по умолчанию должно менять все поля.
"Выход" - выход из приложения.

"Игровое поле"
На игровом поле выдаётся соответствующее режиму уравнение.
Ответ вводится в доступные TextBox поля. Вводятся только числа. Длина ограничена базовыми настройками. Клавиша "Enter" основной клавиатуры может проверить верность решения.
Ответная дробь может быть не приведённой.
Таймер останавливается после верной проверки. Запускается вновь после запроса нового уравнения.
